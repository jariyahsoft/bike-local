# Common Widgets

Shared Flutter widgets that sit above the design-system primitives.

## Planned Shared State Components

- Page skeletons for list, detail, and table screens
- Empty state panel with next-action slot
- Error state panel driven by `error.code`
- Permission denied panel without internal policy leakage
- Offline banner and online-required action guard
- Sync failed panel for queued uploads and ride track recovery
- GPS unavailable panel with permission/device guidance
- Distinct SOS action button and escalation summary card
